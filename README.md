# ❤️ Heart Disease Prediction (ML)

## 📌 Overview
This project implements a Machine Learning model to predict the presence of heart disease using medical dataset features. It focuses only on data processing, model training, and evaluation.

## 🧠 What This Project Does
- Loads and processes dataset  
- Trains ML model(s)  
- Makes predictions  
- Evaluates model performance  

## 📂 Dataset
The dataset includes features like:
- Age  
- Sex  
- Chest pain type  
- Blood pressure  
- Cholesterol  
- Maximum heart rate  

## ⚙️ Tech Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn
